# 📟 Meshtastic Pager OS

![Pager OS concept art](assets/pager-mode-banner.svg)

**Meshtastic Pager OS** — это форк прошивки Meshtastic, сфокусированный на одной задаче: превратить **Heltec V4** в простой приёмник сообщений в стиле пейджера.

Прошивка предназначена для пользователей, которым нужен маленький приёмник, который:

- спокойно ждёт сообщений
- просыпается при входящем сообщении
- крупно и понятно показывает текст
- напоминает о непрочитанных сообщениях
- возвращается в энергосберегающий режим

## 🎯 Типовые сценарии

- техники на объектах
- монтажники, верхолазы, альпинисты
- airsoft / тренировочные группы
- туристические и outdoor-команды
- тяжёлые рабочие условия, где простой пейджер удобнее полноценного ручного устройства

## 🖥️ Экран

OLED делится на три зоны:

1. Верхняя строка: батарея и время
2. Центр: крупный прокручивающийся текст сообщения
3. Нижняя зона: служебная информация пейджера при необходимости

Главный приоритет — зона сообщения. Текст должен быстро читаться, даже если устройство закреплено на одежде или снаряжении.

![Pager OS layout](assets/pager-mode-overview.svg)

## 💬 Поведение сообщений

- Входящее сообщение включает экран
- Текущее сообщение прокручивается по центру
- Новые сообщения добавляются в локальную очередь
- Если сообщение пришло, пока экран уже активен, оно будет показано следующим
- Таймер активности экрана обновляется при новом сообщении
- Непрочитанные сообщения остаются в локальной очереди до подтверждения
- Локальная история временная и может быть очищена

Устройство не должно быть долговременным архивом сообщений. Длинная история должна оставаться в мобильном приложении.

## 🔘 Кнопка

- Короткое нажатие: включить экран / подтвердить сообщение / перейти по непрочитанным
- Долгое нажатие: очистить все локальные сообщения пейджера

Кнопка используется для быстрого подтверждения, а не для полноценной навигации по меню.

## 💡 LED

- Белый LED мигает, пока есть непрочитанные сообщения
- В начале уведомления мигание быстрое
- Потом мигание замедляется для экономии энергии

## 🔗 Совместимость

Сохраняется:

- BLE-подключение и настройка через Meshtastic app
- сообщения каналов
- direct messages
- стандартная логика прошивки и provisioning

## ⚡ Прошивка через браузер

1. Открыть [Pager OS Web Flasher](https://chemobook.github.io/meshtastic-pager-mode/)
2. Использовать desktop Chrome или Edge
3. Выбрать **Heltec V4**
4. Прошить из браузера
5. Дождаться перезагрузки

## 🧪 Статус

- Проект экспериментальный
- Это не официальный релиз Meshtastic
- Требуется тестирование на реальном устройстве
