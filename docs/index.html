Meshtastic Pager OS Web Flasher

Browser installer for a standalone pager-style firmware fork built on top of the Meshtastic radio stack. The idea is simple: receive messages, read them fast, confirm them, go back to low-power behavior.

Веб-прошивальщик для отдельной pager-прошивки на базе Meshtastic. Идея простая: принять сообщение, быстро его увидеть, подтвердить и снова вернуться в экономичный режим.

The public browser installer is intentionally narrow right now. It exposes one primary target: Heltec V4. That keeps the page honest and avoids flashing the wrong image to the wrong board.

Heltec V4

ESP32-S3 16MB flash Latest packaged build

Main target for the current Pager OS concept. Use this only for boards that must be flashed with the `heltec-v4` environment.

  • Full browser install through Web Serial
  • Uses the packaged images from `release-work/firmware/heltec-v4`
  • After flashing, configure the node in the standard Meshtastic mobile app
Package files: release-work/firmware/heltec-v4

How to flash

  1. Use current desktop Chrome or Edge.
  2. Connect the board with a real data USB cable.
  3. Close any terminal or serial tool that may hold the port.
  4. Click the install button and choose the board serial port.
  5. Wait for the browser installer to finish and allow the reboot.
  6. Open the standard Meshtastic mobile app and configure the node over BLE.

Browser flashing depends on Web Serial, so Safari and most mobile browsers are not the primary path.

Как прошивать

  1. Используй актуальный настольный Chrome или Edge.
  2. Подключи плату нормальным USB-кабелем для передачи данных.
  3. Закрой терминалы и serial-программы, которые могут держать порт.
  4. Нажми кнопку прошивки и выбери serial-порт устройства.
  5. Дождись завершения браузерной прошивки и перезагрузки.
  6. Открой обычное приложение Meshtastic и настрой узел по BLE.

Веб-прошивка зависит от Web Serial, поэтому Safari и большинство мобильных браузеров здесь не основной путь.

Before flashing

  • This is an unofficial community firmware fork.
  • The public installer currently targets Heltec V4 only.
  • Use the correct image for the correct board.
  • If no serial port appears, replace the USB cable first.
  • If the port exists but cannot be opened, close other serial tools.
  • If Windows does not see the board, install the required ESP32 USB serial driver.
  • If the install fails once, retry from the same page before switching tools.
