How to flash

  1. Use a desktop browser with Web Serial support: Chrome or Edge.
  2. Connect the device with a real data USB cable.
  3. Close other serial tools before starting the install.
  4. Choose the exact board, click the install button, then pick the serial port.
  5. Wait for the browser flasher to finish and let the device reboot.
  6. After first boot, configure the node in the Meshtastic app as usual.

Browser flashing requires Web Serial. For the fewest problems, use current desktop Chrome or Edge.

Как прошивать

  1. Используй настольный Chrome или Edge с поддержкой Web Serial.
  2. Подключи устройство нормальным USB-кабелем для передачи данных.
  3. Перед прошивкой закрой другие программы, которые держат COM-порт.
  4. Выбери точную модель платы, нажми кнопку прошивки и укажи serial-порт.
  5. Дождись завершения прошивки и перезагрузки устройства.
  6. После первой загрузки настрой узел через обычное приложение Meshtastic.

Для веб-прошивки нужен Web Serial. Самый надёжный вариант сейчас: настольный Chrome или Edge.

Important notes

  • This page is for `heltec-v3` and `heltec-v4` only.
  • Choosing the wrong board can leave the device in a bad state.
  • This is an unofficial community fork, not an upstream Meshtastic release.

Troubleshooting

  • If no serial port appears, first replace the USB cable with a known data cable.
  • If Windows does not see the board, install the required ESP32 USB serial driver.
  • If Linux sees the board but the browser cannot open it, check serial permissions and close terminal tools.
  • If the install fails once, retry from the same page before switching to another tool.
  • If the wrong target was selected, stop and re-flash only with the correct board package.
