# Release Title

`Meshtastic Pager Mode Fork 2.7.23.e214feb for Heltec V3 / V4`

## Short Description

Unofficial community Meshtastic fork for small-screen devices with Pager Mode improvements, prepared for `heltec-v3` and `heltec-v4`. This project was created by interested people without financial motivation, and a large part of the work was done with AI assistance because the maintainer works in a different field.

## Changelog Bullets

- Added and refined Pager Mode behavior for compact-screen usage
- Reduced unnecessary battery drain while pager mode is active
- Pager mode now restores its focused session after reboot
- Long press now opens message actions for pager-mode control
- Pager-mode autoscroll no longer runs into the header or creates blank gaps while scrolling
- Prepared ready firmware packages for `heltec-v3`
- Prepared ready firmware packages for `heltec-v4`
- Added bilingual project and release documentation
- Added a friendlier fork-specific flash helper for `Heltec V3 / V4`
- Kept the fork openly marked as unofficial, community-made, and AI-assisted
- Testing is still ongoing on real devices
